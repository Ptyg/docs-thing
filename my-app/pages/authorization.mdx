# Authorization

<Callout type="caution">
  All API requests require authentication. Follow the instructions below to authorize your requests.
</Callout>

## API Key Authentication

To authenticate API requests, include your API key in the `Authorization` header:

```bash
Authorization: Bearer YOUR_API_KEY
```

Replace `YOUR_API_KEY` with your actual API key.

## Example Request

Hereâ€™s an example of an authenticated request using the SDK:

```ts
import { Configuration, ClientsApi } from '@your-org/crm-sdk';

// Set up the configuration with your API key
const config = new Configuration({
  apiKey: 'YOUR_API_KEY',
});

// Initialize the Clients API with the configuration
const clientsApi = new ClientsApi(config);

// Fetch all clients
clientsApi.getClients()
  .then((clients) => {
    console.log('Clients:', clients);
  })
  .catch((error) => {
    console.error('Error fetching clients:', error);
  });
```

<Callout type="caution">
  Keep your API key secure and do not expose it in client-side code.
</Callout>
