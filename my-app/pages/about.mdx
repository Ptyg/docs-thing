# About

<Callout type="info">
  This project provides a REST API for managing clients and invoices in a CRM system.
</Callout>

## Project Overview

The **CRM System API** is designed to help you manage:

- **Clients**: Create, read, update, and delete client records.
- **Invoices**: Manage invoices associated with clients.

The API is built using **OpenAPI 3.0** and documented with **Zudoku**.
